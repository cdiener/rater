How does it work?

  • Applicants can be reviewed by clicking on Applicants in the menu bar. Simply go ahead an review as many applicants as you can.
  • You can see the latest results by clicking on Results in the menu bar.
  • In case you are allowed to do so, you can also review abstracts by clicking on the Abstracts link in the menu bar (if you don't see the link you are not allowed to ).
  • In case you are allowed to do so, you can upload new applicant and abstract data by clicking the Import link. Please check with Chris D. first whether there is a recent backup before doing so.
  • 5
      this bubble denotes the number of reviews the applicant/abstract has already received. This way you can check easily whether there have been previous reviews.
